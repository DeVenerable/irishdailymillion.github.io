
# Irish Daily Million Full Archive Generator

Files included:
- irish_daily_million_scraper.py
- sample_archive_template.html

The execution environment used to generate these files does not have internet
access, so the complete live archive cannot be downloaded directly here.

To generate the full archive locally:

1. Install dependencies:

   pip install requests beautifulsoup4

2. Run:

   python irish_daily_million_scraper.py

This will generate:
- irish_daily_million_archive.txt
- irish_daily_million_archive.html

in a MerseyWorld-style layout.
